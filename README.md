# E-commerce Sales Dashboard (Excel)

## Project Overview
This project is an **E-commerce Sales Analysis Dashboard** created using **Microsoft Excel**.  
The dashboard provides a clear and interactive view of sales performance, profitability, order trends, and category-wise insights to support data-driven business decisions.

## Objectives
- Analyze overall sales, profit, quantity sold, and number of orders  
- Track **Year-over-Year (YoY) growth** for key business metrics  
- Understand category-wise and sub-category-wise performance  
- Identify top-performing regions and products  

## Key Features
- **KPI Cards**: Sales, Profit, Quantity, Orders, and Profit Margin with YoY growth
- **Sales & Profit Trend Analysis** (Monthly)
- **Category-wise Sales and Profit Report**
- **Category-wise Sales Share (%)**
- **Top 5 Sub-Categories by Sales**
- **State-wise Sales Map**
- **Interactive Filters**:
  - Year
  - Region
  - Segment (Consumer, Corporate, Home Office)

## Tools & Skills Used
- Microsoft Excel  
- Pivot Tables  
- Pivot Charts  
- Excel Formulas  
- Data Cleaning  
- Data Analysis  
- Dashboard Design
